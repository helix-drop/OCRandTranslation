D — 导航 + 状态 + 行内编辑 （推荐）

左侧导航每项带小圆点表示配置状态和计数。右侧 Providers 直接行内输入 key + 即时状态 pill，不用每个 Provider 一张大 card。一次"统一保存"，一键解决。

4 版共同点：PaddleOCR / DeepSeek / DashScope / GLM / Kimi / MiMo 的 Key 从"6 张独立 card + 6 个独立表单"收敛为行式列表；模型池槽位从"大卡片嵌套"简化为紧凑编辑；术语词典 / 目录 / 数据管理按需分组。
